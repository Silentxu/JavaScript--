# 工厂方法
通过该方法可以大批量的创建对象
```js
function createPerson(name,gender,age){
    //先创建一个新的对象
    var obj = new Object;
    obj.name = name;
    obj.gender = gender;
    obj.age = age;
    obj.sayName = function(){
        alert(this.name);
    }

    //将新的对象返回
    return obj;
}

var obj2 = createPerson();
var obj3 = createPerson();
var obj4 = createPerson();
```
使用工厂方法创建的对象，使用的构造函数都是object
所以创建的对象都是object这个类型
就导致我们无法区分出多种不同类型的对象
